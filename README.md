Program Biaya Pengiriman (Ekspedisi)Fungsi (Tujuan):Menghitung total biaya pengiriman paket berdasarkan tiga faktor: berat, jarak tempuh, dan volume.
Cara Kerja:Input: Meminta pengguna memasukkan berat (kg), jarak (km), serta dimensi (panjang, lebar, tinggi dalam cm).Hitung Volume: Menghitung volume ($P \times L \times T$).Logika Biaya Jarak (if-else): Mengecek jarak. Jika $\le$ 10 km, biaya per kg adalah Rp 4.250. Jika $>$ 10 km, biayanya Rp 6.000. Hasilnya dikalikan dengan berat.Logika Biaya Volume (if): Mengecek volume. Jika $>$ 100 $cm^3$, menambahkan biaya tetap Rp 50.000.Output: Menjumlahkan "Biaya Jarak" + "Biaya Volume" dan menampilkannya sebagai total biaya.

Program Cek Ganjil Genap Fungsi (Tujuan): Menentukan apakah satu bilangan bulat yang dimasukkan adalah bilangan ganjil atau genap.

Proses (Modulo): Menggunakan operasi Modulo (%) untuk mencari sisa bagi bilangan tersebut dengan 2.Logika (if-else): Jika bilangan % 2 hasilnya 0 (habis dibagi 2), program mencetak GENAP.Jika hasilnya bukan 0 (sisa 1 atau -1), program mencetak GANJIL.

Program Hitung FaktorialFungsi (Tujuan):Menghitung nilai faktorial ($n!$) dari sebuah bilangan, yaitu perkalian semua bilangan positif dari 1 sampai bilangan tersebut (misal: $4! = 1 \times 2 \times 3 \times 4 = 24$).
Cara Kerja:Input: Meminta pengguna memasukkan satu bilangan bulat ($n$).Inisialisasi: Menyiapkan variabel hasil dengan nilai awal 1.Proses (Loop): Menggunakan for loop untuk berulang dari i = 1 sampai i = n.Akumulasi: Di dalam setiap perulangan, nilai hasil diperbarui dengan hasil = hasil * i. (Jadi, $1 \times 1$, lalu $1 \times 2$, lalu $2 \times 3$, lalu $6 \times 4$).Output: Menampilkan nilai akhir hasil.

Program Volume Tabung (dengan Fungsi)Fungsi (Tujuan):Menghitung volume tabung ($\pi \times r^2 \times t$) dengan memisahkan logika perhitungan ke dalam sebuah fungsi (method) terpisah.
Cara Kerja:Input (di main): Program utama (main) meminta input jari-jari ($r$) dan tinggi ($t$).Pemanggilan Fungsi: Program main memanggil fungsi hitungVolume() dan mengirimkan nilai $r$ dan $t$ tadi sebagai parameter (umpan).Proses (di fungsi): Fungsi hitungVolume() menerima $r$ dan $t$, lalu menghitung $\pi \times r^2 \times t$.Pengembalian Nilai: Fungsi tersebut mengembalikan (return) hasil perhitungan kembali ke main.Output (di main): Program main menerima hasil itu dan menampilkannya ke pengguna.

Program Rata-rata Nilai SiswaFungsi (Tujuan):Menghitung nilai rata-rata dari sejumlah nilai ujian siswa, di mana jumlah siswanya ditentukan oleh pengguna.
Cara Kerja:Input Awal: Meminta pengguna memasukkan jumlah siswa (misal $n=5$).Inisialisasi: Menyiapkan variabel totalNilai = 0.Proses (Loop): Menggunakan for loop untuk berulang sebanyak $n$ kali (5 kali).Input Berulang: Di dalam loop, program meminta "Masukkan nilai siswa ke-..." dan langsung menambahkannya ke totalNilai.Kalkulasi: Setelah loop selesai, program menghitung rata-rata dengan rumus rataRata = totalNilai / jumlahSiswa.Output: Menampilkan rataRata.

Program Penjumlahan Matriks Fungsi (Tujuan): Menjumlahkan dua buah matriks (Matriks A dan Matriks B) yang elemennya dimasukkan oleh pengguna.
Cara Kerja:Input Dimensi: Meminta pengguna memasukkan jumlah baris dan kolom.Input Matriks A: Menggunakan perulangan bersarang (nested loop) untuk meminta pengguna memasukkan setiap elemen Matriks A (misal: A[0][0], A[0][1], A[1][0], dst.).Input Matriks B: Mengulangi proses yang sama untuk Matriks B.Proses Penjumlahan: Menggunakan nested loop ketiga. Di setiap posisi [i][j], program menghitung Hasil[i][j] = A[i][j] + B[i][j].Output: Mencetak Matriks Hasil (dan biasanya juga Matriks A dan B) ke layar secara rapi.
